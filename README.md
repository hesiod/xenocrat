# xenocrat [![Build Status](https://travis-ci.org/hesiod/xenocrat.png)](https://travis-ci.org/hesiod/xenocrat)

Xenocrat is an orbital mechanics simulator written in Haskell.

## Install
    cabal sandbox init
    cabal install -j

## Run
    .cabal-sandbox/bin/xenocrat
